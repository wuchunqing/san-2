About This Tool

Whether you are planning a road trip, reading a hiking trail map, or working with marine charts, distances appear in different units around the world. This travel distance meter converts any value using internationally defined length factors, with meters as the internal reference. All six supported units are shown together so you can compare results at a glance.

Distance Units Explained

Kilometer (km) Metric unit equal to 1,000 meters. Common for road distances and travel planning worldwide.
Meter (m) SI base unit of length. Used for short distances, athletics tracks, and scientific work.
Mile (mi) International statute mile = 1,609.344 m. Standard for US and UK road distances.
Foot (ft) Imperial unit = 0.3048 m exactly. Often used for elevation and smaller measurements.
Yard (yd) Imperial unit = 3 feet = 0.9144 m. Used in US sports fields and some surveying.
Nautical mile (nmi) Marine/aviation unit = 1,852 m. One minute of latitude along a meridian.

How to Use

  • Enter a distance value and choose the Input Unit.
  • Select the Target Unit for the main conversion shown above.
  • Review All Unit Conversions for every supported unit.
  • Use Copy to copy the primary conversion and all unit values.
  • Clear resets to the default example (10 km → mi) and recalculates.
